# Slagerij & Traiteur Benoît en Veerle

Website voor Slagerij & Traiteur Benoît en Veerle in Huldenberg. Gebouwd met Next.js, Tailwind CSS en Resend voor e-mails. Gedeployed op Vercel.

## Ontwikkeling

```bash
npm run dev
```

## Stack

- Next.js 16 (App Router)
- Tailwind CSS
- Resend (e-mail)
- Vercel (hosting)
